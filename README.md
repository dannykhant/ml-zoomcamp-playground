# ml-zoomcamp-playground
This is the playground for Machine Learning Zoomcamp...

#### Dependencies:
* uv
* numpy
* pandas
* scikit-learn
* seaborn
* jupyter

```bash
uv add numpy pandas scikit-learn seaborn jupyter
```

#### To run Jupyter-notebook in Codespaces
```bash
jupyter notebook --no-browser --port 8888
```
